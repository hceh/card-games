# Card Games
This project looks to create the basis from which basic card games can be built
in Python. 

## Current Features
- `src.card_deck`
    - has generic classes for a Card, CardDeck, Table and Hand
- `src.rummy`
    - super classes of the generics in `src.card_deck` specific to the game 
    [Rummy](https://bicyclecards.com/how-to-play/rummy-rum/) with Aces high
